MLB Advanced Analytics Platform

  • Project Overview
    • Full-stack baseball analytics web app deployed live on Vercel (frontend) and Railway (backend).
    • Aggregates data from MLB Stats API, Baseball Savant (Statcast), and Baseball Reference WAR CSVs — no paid API required.
    • Features player profiles, leaderboards, spray charts, pitch zone heatmaps, team analytics, and contract value analysis.
  • Custom Statistic: Park Favorability Index (PFI)
    • Designed a composite 0–200 score quantifying how each ballpark affects batters and pitchers.
    • Built from 3-year rolling home/away splits (runs, HR, BABIP, K, BB, EV) fetched live from the MLB Stats API.
    • Pitcher PFI = 200 − Batter PFI, giving a mirrored scale for both sides.
  • Tech Stack
    • Frontend: React 18, TypeScript, Vite, Tailwind CSS, Recharts, React Query
    • Backend: Python 3.11, FastAPI, pandas, httpx — Parquet disk cache + in-memory TTLCache
    • Deployment: Railway (Docker container), Vercel (static frontend)
  • Key Features
    • Statcast metrics: xBA, xSLG, xwOBA, Barrel%, Hard Hit%, Exit Velocity, Launch Angle
    • Platoon splits (vs LHP/RHP, Home/Away) with deduplication for mid-season trades
    • Player comparison radar charts, year-by-year career trends, spray charts
    • Team Historical Analytics: WAR/wRC+/FIP vs win% correlations across single seasons or all-season aggregates (2015–present)
