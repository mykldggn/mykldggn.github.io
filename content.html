Soccer Computer Vision Project

Based on DFB-Pokal game data from Roboflow, this project utilizes the Roboflow/HF API for advanced soccer analytics. The project covers object detection to track players and the ball, homography fundamentals to map camera views to a standard pitch, and K-means clustering to analyze player positioning patterns.

The result is a Voronoi diagram of the game clip, showing the areas of control for each player, along with a ball tracking diagram that visualizes movement patterns. This type of analysis can provide coaches and analysts with insights into team tactics, space utilization, and player movement that might not be visible to the naked eye.

Credit Card Fraud Deep Learning Neural Network

This notebook analyzes a highly unbalanced Kaggle Credit Card Fraud dataset. The project compares different classifiers for an undersampled versus an oversampled (SMOTE) neural network, leveraging Keras and TensorFlow frameworks.

Given the extreme imbalance in fraud detection datasets (with fraud cases typically less than 1% of transactions), the notebook explores techniques to address this challenge. By comparing various approaches, it demonstrates the effectiveness of different sampling methods in improving fraud detection accuracy without sacrificing precision.
